---
name: greeting
description: Generate a warm, friendly greeting message for a person. Use when the user asks for a greeting or wants to be welcomed.
argument-hint: <name>
---

When this skill is invoked with $ARGUMENTS:

1. Extract the name from the arguments
2. Generate a warm, friendly greeting that includes their name
3. Make the greeting welcoming and personable
4. You can vary the greeting style to keep it interesting (but keep it appropriate and professional)
5. If no name is provided, ask for one

## Examples

```
User: /greeting Sarah
Result: Hello, Sarah! Welcome to the Simple Agent Demo! It's great to have you here.
```
